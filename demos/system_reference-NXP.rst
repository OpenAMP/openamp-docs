====================
NXP Reference Boards
====================

A number of the
`OpenAMP project examples <https://github.com/OpenAMP/openamp-system-reference/tree/main/examples>`_
can be executed on NXP Reference boards.

* NXP `8MPLUSLPD4-EVK <https://www.nxp.com/design/design-center/development-boards-and-designs/8MPLUSLPD4-EVK>`_.

The `AN13970 Running Zephyr RTOS on Cadence Tensilica HiFi 4 DSP <https://www.nxp.com/docs/en/application-note/AN13970.pdf>`_
application note provides instructions for setting up the evaluation board.
