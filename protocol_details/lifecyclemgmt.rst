========================================
Life Cycle Management through RemoteProc
========================================

The LCM (Life Cycle Management) component of OpenAMP is known as remoteproc. Remoteproc APIs
provided by the OpenAMP Framework allow software applications running on the host processor to
manage the life cycle of a remote processor and its software context. A complete description of the
remoteproc workflow and APIs are provided.
